hoover Documentation
********************

This is the documentation for hoover.

``hoover`` is a package used to calculate likelihoods for Galactic foreground
models. Given a set of observations in pixel-space, the corresponding
pixel variances, and a given spectral model, ``hoover`` calculates the
posterior densities on specified spectral parameters and amplitudes.
